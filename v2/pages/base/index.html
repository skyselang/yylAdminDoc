基础

安装之前请确保PHP环境和Vue环境已经搭建好并且可以运行。

# 开发环境

  • phpStudy (opens new window)（Windows10，Nginx1.16，MySQL5.5，PHP7.2，Redis）
  • Composer (opens new window)（v2.x）
  • Node.js (opens new window)（v12.x）

# 版本控制

  • Git (opens new window)

# 后端框架、插件

  • Apidoc (opens new window)

# 前端框架、UI

  • Vue (opens new window)（v2.x）
  • Element (opens new window)（v2.x）
  • vue-element-admin (opens new window)
