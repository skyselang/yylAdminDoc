## 简介

yylAdmin是一个基于ThinkPHP6和Vue2极简后台管理系统，只有登录退出、权限管理、日志管理、接口文档与调试等基础功能；前后台基础框架；你可以在此基础上根据你的业务需求进行开发扩展。前后分离，简单轻量，免费开源，开箱即用。

## 功能

- 登录退出
- 权限管理
- 日志管理
- 实用工具
- 接口文档：自动生成接口文档，在线调试
- 更多动能请安装后体验...
