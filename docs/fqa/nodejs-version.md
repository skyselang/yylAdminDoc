# Node.js 版本问题

## node.js

node.js 版本兼容 >= 18  
查看当前 node.js 版本：

```bash
node -v
```

## nvm

[nvm](https://github.com/coreybutler/nvm-windows)（Node Version Manager）是一个功能强大的 Node.js 版本管理工具，它允许用户在同一台电脑上安装和管理多个 Node.js 版本，从而轻松地在不同项目间切换所需的 Node 环境。
