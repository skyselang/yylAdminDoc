::: warning 注意
开启调试模式`DEBUG`且启用apidoc代码生成器配置，确保站点目录有777权限，一般在本地开发时使用。  
.env  
APP_DEBUG = true  
[APIDOC]  
GENERATOR_ENABLE = true  
:::

代码生成器功能使用的是 [ThinkPHP-ApiDoc](https://gitee.com/hg-code/thinkphp-apidoc)。  
配置+模板即可快速生成代码及数据表的创建。
